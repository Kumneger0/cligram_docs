# Stories

- **View Stories**: Press `alt + s` to open the stories modal.
- **Story Indicator**: A number above the chat list in the sidebar indicates the number of available stories.
- **Navigate Stories**: Use the arrow keys to navigate through the list of users with stories.
- **View a User's Story**: Press `enter` to view the stories of the selected user.
